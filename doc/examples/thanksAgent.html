ThanksAgent

Author: Fabio Bellifemine (TILab S.p.a)
Date: 31/08/2001

This example proposes an agent that creates some other agent and has a simple conversation with them. Notice how the agent registers itself with the Directory Facilitator.

 

ThanksAgent
Description:
This agent implements the following functionality:
 - When being born one will register
 - It will create a list of agents registered with the DF.
 - It will send a message to them of greeting
 - It will listen if there is some answer
 - To the agents who answer it will thank to them to answer.
Jade classes used:
CyclicBehaviour, DFServiceCommunicator,DFServiceDescription,DFAgentDescription

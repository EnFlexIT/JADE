JADE Administrative Tutorials

Author: David Grimshaw (Ryerson University)

Last Modified: July 12, 2004

Java Platform: Java 2 Standard Edition for Windows, 1.4 or later.

JADE version: 3.2 or later.

This set of tutorials is designed to supplement the JADE Administrative Guide. They are based on my own experience learning to run JADE. I hope they will smooth the learning curve for students and other people wishing to use JADE. Once you get past the administrative details of setting up JADE in an appropriate configuration you can get on to the real fun of programming Multi-Agent Systems using JADE.

Tutorials

  1. Running JADE with one (MAIN) container.
  2. Using more than one container
  3. Running multiple JADE platforms using an IIOP MTP.
  4. Using the HTTP Message Transport Protocol.
  5. Using JBuilder with JADE.

JADE Administrative Tutorials

JADE comes in a zip file. Just unzip it, preserving the directory structure. On Windows systems you usually put it in c:\jade.

The Administrative Guide.

This is the main reference for running JADE. The Administrative Guide is in Adobe pdf file format. It is available on line, along with all the other JADE documentation at http://sharon.cselt.it/projects/jade/. It also comes with the JADE distribution. The guide is quite detailed.
